== Video Library Readme

These are the steps to get this application up and running:

* Ruby version 2.2.1p85

* System dependencies
  * Ruby 2.2.1p85
  * Mysql
  * Rails 4.2.4

* Configuration
After downloading and unzipping the files and installing the Bundler gem cd into the directory and run bundle install.

* Database creation and initialization
rake setup

* How to run the test suite
rspec
